# challenge-field-validator

Scale AI Challenge

## Usage

Install dependencies

```bash
pnpm install
```

Build the project

```bash
pnpm build
```

Run the cli

```bash
pnpm start
```

Run the cli with a specific script

```bash
pnpm start -s <script-name>
```

Run the cli and pipe the output to a file while running the script

```bash
pnpm start | tee output.txt
```
